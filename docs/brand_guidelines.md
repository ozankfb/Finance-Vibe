# Brand Guidelines (Source of Truth)

## Style keywords
Clean, minimal, modern, confident.
No unnecessary decoration.

## Color + Theme
App supports Light and Dark mode.
Primary accent is a bright/neon green used for:
- Primary buttons
- Active states
- Key highlights

Neutrals:
- Backgrounds are clean and low-contrast
- Text is high-contrast and readable

## Typography
- Simple system-like typography
- Clear hierarchy: Title, Body, Caption
- Avoid fancy fonts and excessive weights

## Layout rules
- Spacious vertical rhythm
- Consistent padding and spacing
- Cards use subtle separation, not heavy borders
- Rounded corners used consistently

## Components
Prefer reusable primitives:
- Button (primary/secondary/ghost)
- Input
- OTP input (6 digits)
- Card
- Divider
- Screen container

## Do / Don’t
Do:
- Keep UI quiet and intentional
- Use the green accent with restraint
Don’t:
- Add gradients, glows, random shadows
- Invent new patterns that are not in the specs
