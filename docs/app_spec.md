# App Spec (High level)

## What this app is
A mobile investing/portfolio style app with onboarding + auth and 3 main tabs: Home, Timeline, Market.

## Screens
Auth Flow:
1. Splash
2. Launch / Login
3. Country Picker
4. Phone Number
5. OTP Verification
6. Success

Main App (Tabs):
7. Home
8. Timeline
9. Market

## Core UX notes
- The auth flow is phone-based (country + phone + OTP)
- After success, user enters the main tab experience
- UI must support light and dark themes consistently
- Reuse components across screens
